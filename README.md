# FitEats 🍱🥗

### A Smart Food Ordering Web App with Nutritional Transparency

FitEats is a modern food ordering platform that goes beyond traditional delivery apps by helping users make healthier and smarter food choices. Unlike regular food delivery platforms, FitEats provides detailed nutritional information such as calories, protein, fat, fiber, and carbohydrates for every meal. 

---

## 🚀 Features

### 🥗 Nutritional Information Display

* View calories, protein, fat, fiber, and other nutrition details for every dish
* Helps users maintain fitness goals and dietary preferences
* Supports healthier decision-making while ordering food

### ⭐ Review & Rating System

* Customers can rate restaurants and dishes
* Read authentic customer reviews before ordering
* Improves trust and food quality transparency

### 👤 User Profiles & Preferences

* User authentication and profile management
* Save favorite dishes and restaurants
* View order history and personalized recommendations

### 🛒 Smart Cart System

* Add/remove food items easily
* Displays total nutritional values of the cart
* Promo code support and order summary

### 🚚 Delivery Management

* Dedicated delivery agent dashboard
* Real-time active order tracking
* Order status updates and navigation support

### 🍽️ Restaurant Admin Dashboard

* Manage dishes and menus
* View orders and restaurant analytics
* Add, update, or remove food items

---

# 🧠 What Makes FitEats Different?

Most food delivery platforms focus only on fast ordering and convenience.
FitEats focuses on both **convenience + health awareness** by providing:

* Nutritional transparency
* Fitness-oriented meal discovery
* High-protein / low-carb / fiber-rich food filtering
* Better food decisions for health-conscious users

# 📦 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/fiteats.git

# Move into project folder
cd fiteats

# Install frontend dependencies
cd frontend
npm install

# Install backend dependencies
cd ../backend
npm install
```

---

# ▶️ Run the Project

## Frontend

```bash
npm run dev
```

## Backend

```bash
npm start
```

---

# 🌐 Environment Variables

Create a `.env` file in the backend directory and add:

```env
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```
